# 🌿 Классификация состояния растений по мультиспектральным снимкам с дронов

**Автор:** Анна Герасимова, группа ИТЗ2441, КубГАУ

## 📝 Описание проекта
Проект по классификации состояния растений (здоровое/больное, стресс от засухи/удобрений) с использованием мультиспектральных снимков, полученных с дронов.  
В основе лежат алгоритмы из репозитория [AI-Agro](https://github.com/RentadroneCL/AI-Agro), адаптированные под задачу бинарной/многоклассовой классификации.

## 🎯 Цель и мотивация
Раннее выявление болезней растений позволяет снизить потери урожая и оптимизировать применение удобрений. Мультиспектральная съёмка с дронов даёт возможность обнаружить стресс до появления видимых симптомов.

## 📊 Используемые данные
- **PlantVillage Dataset** – изображения здоровых и больных листьев (15+ классов)
- **Источник**: [Kaggle - PlantVillage](https://www.kaggle.com/datasets/emmarex/plantdisease)

## 🏗️ Архитектура модели
- **EfficientNet-B4** – оптимальный баланс точности и скорости
- **ResNet-50** – для сравнения
- Расчёт вегетационных индексов (NDVI, NDRE) из мультиспектральных каналов
